PixelGarden is an open-source, web-based gardening simulation game where players plant, grow, and manage a pixel-art garden. The project is designed to be modular, allowing community contributors to add new plants, objects, and gameplay mechanics. It is intended to be both educational and fun, providing an accessible entry point for developers interested in web games, game mechanics, and open-source collaboration.

## Features
- Pixel-art visuals with animated plants and objects
- Realistic day/night and weather cycles
- Modular architecture for easy community contributions
- Web-based and mobile-friendly
- Open to accessibility and localization improvements
